proCalc API Reference

Programming a button for this calculator is very simple, all you have to do, is write standard JavaScript code. In order to have access to the values the user have typed, you will use some of the functions in the API. In order to control certain features of the calculator, or to instruct the calculator to display your results you will make use of other functions of this API.

It is a good idea to look at the code included in the calculator. Yes, most of the built in functionality is implemented in the same way you will use to customize it. As a result, you can modify, hide or delete almost everything. The only exeption are the number buttons the basic opperations and the memories.

Most of the functions you will write will operate on a single number e.g. Sin, Cos, n!, etc. Sometimes, however, you may want to operate on 2 or more numbers for example y to the power x. For these ocations you will write code under 'onEqual (=) code:' this code will run when the user hits the '=' button. Take a look at the included code for 'y to the power x', or 'x root of y'.

 

co - CalculatorObject:

It is the base object for calculator related work.
Returns the calculator object


get(int level)

Gets you any level of calculator value.
Returns a String
Example:
given --> 2 + 3 - 4
co.get(0) will return "4"
co.get(1) will return "3"
co.get(2) will return "2"
co.get(3) will return ""
co.getX() will return "4"


getX()

Gets you the current value of X. (convenience method)
Returns a String
Example: see 'get()'


appendX(String value)

Will put 'value' at the end of the display. It will also, internally, add it to X.
Does not return a value.


setX(String value)

Will put 'value' at the end of the display substituting the value of X. It will also, internally, add it to X.
Does not return a value.
Example:
Given: 4 + 3
You now want to add 5 squared.
You type + 5 x 2
If you use 'appendX()' you would end up with:
4 + 3 + 525
25 being your result and 5 the number you had typed (X)
'setX()' will make 25 replace 5 (the current X) leaving you with the desired
4 + 3 + 25

setD(String value)

Will put 'value' at the end of the display. The value of X remains unaffected.
Does not return a value.

 

putFlag(String flag)

Sets a user defined flag that persists over calls to your code.
Does not return a value.
Example:
Let's say that your code starts a timer. You can set a flag "timer". Next time the same code runs, it can check the timer flag to determine if the timer has already been started.

If(co.peekFlag("timer")){
//Code to stop the timer and display the result
co.takeFlag("timer");
} else {
//Code to start timer
co.putFlag("timer");
}

 

peekFlag(String flag)

Will return true if the flag exists and false if it doesn't. It will not affect the existing flag
Returns a boolean

 

takeFlag(String flag)

Will return true if the flag exists and false if it doesn't. It will also remove the flag
Returns a boolean


T(String mssg)

This function will display a standard toast with the message you pass it.
Does not return a value.
Example:
co.T("Hello World!");


dialogTitle(String title)

Sets the title of a dialog box to be shown.
Does not return a value.

 

dialogHeading(String heading)

Sets the heading of a dialog box to be shown.
Does not return a value.


dialogContent(String content)

Sets the contents of a dialog box to be shown.
Does not return a value.


dialogBtnPositive(String btnText)

Sets the text of the button in a dialog box to be shown.
Does not return a value.


void showDialog()

Shows a previously prepared dialog box.
Does not return a value.
Example:
co.dialogTitle("proCalc Dialog");
co.dialogHeading("Show your result in a dialog box");
var ctnt = "Sometimes you may need more room to show your results than proCalc's display can handle. You can follow this template to show your results in a dialog box.";
ctnt += "\n";
ctnt += "\n";
ctnt += "For example";
ctnt += "\n";
ctnt += " Your x value is: ";
ctnt += co.get(0);
ctnt += "\n";
ctnt += " Your y value is: ";
ctnt += co.get(1);

ctnt += "\n";
ctnt += "\n";
ctnt += "\n";

ctnt += "Enjoy :)";
co.dialogContent(ctnt);
co.dialogBtnPositive("OK");
co.showDialog();
